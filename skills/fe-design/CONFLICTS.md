Source-of-truth record for how contradictions between the merged skills were resolved. Locked with the user 2026-06-12. When re-merging after an upstream update, re-check each row. New unresolved collisions: ask the user once, then add a row here.

**Global precedence:** design-impeccable / emil-design-eng / design-taste-frontend-v2 (the trio) always beat gpt-taste, high-end-visual-design, stitch-design-taste, minimalist-ui, industrial-brutalist-ui, taste-v1. Within the trio: domain ownership (impeccable = layout/spacing/typography/color/copy · emil = motion · taste-v2 = landing direction, bans, design-system map), and for bans, strictest wins.
